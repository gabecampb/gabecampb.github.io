Why?

MABS was created as a way for me to develop platform-independent software as hobby projects.

Limitations

There are several limitations of MABS, notably:
- Primary/secondary/output register-number referencing system that must be updated between instructions when necessary (result of tiny instruction width)
- No support for geometry, mesh, tessellation, or intersection shaders
- No support for AABB geometries in ray tracing
- No support for accessing data from multiple VBOs in a single draw call
- No support for indirect draw calls
