This is a bytecode set designed for efficient interpretation in virtual machines, consisting of 8-bit instructions.

Current version: 0.21

Features

* Keyboard input
* Mouse input
* Touchscreen input
* Graphics API (compatible with modern and previous generation graphics APIs)
* A shading language, represented in bytecode
* GPU computing
* Multi-threading and a high level of control over threads
* File I/O, including recognition and selection of connected drives
* SIMD
* Controller and HMD support
* Image capture from display or cameras
* 2D/3D audio
* Sound transmission, absorption, and scattering
* Audio file creation and editing
* Audio recording
* Audio playback
* Video file creation and editing
* Video playback
* Ray tracing
* Networking

Future of the Project

In the future, I will continue to clean up the specification as much as possible, involving elaboration, reorganization, and making it more concise. Many minor changes to the specification will be made, but the instructions will almost certainly not be changed. I have been working on a virtual machine since the release of version 0.12, and it is mostly complete for version 0.19 but some of the code must now be updated for version 0.21. Forward compatibility for MABS programs will be ensured from version 1.0 onward, which I hope to release soon.
